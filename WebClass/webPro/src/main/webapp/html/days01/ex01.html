html 수업 1일차

  • html
    1. HyperText Markup Language
    2. 웹페이지의 구조를 만드는 마크업 언어
    3. 태그(Tag)를 사용해서 문서 구조를 정의: 시작 태그, 닫기 태그
    4. Element(요소) = 시작태그+내용(Content)+닫기태그
  • HTML 기본 구조
    1. <!DOCTYPE html> 문서 유형 선언문, html5 문법따르는 문서, 대문자 권장
    2. html : html 문서의 시작 ~ 끝, 루트 (root)엘리먼트, 모든 요소의 최상위 부모요소
    3. html > head : html 문서의 메타 정보가 포함. 필수가 아님.
    4. html > head > meta :인코딩 UTF-8 메타 정보
    5. html > head > title : 브라우저의 제목 지정
    6. html > body : 필수 o , 문서의 본문을 정의(제목, 이미지, 표, 목록 등등)을 정의하고 웹페이지에 보여지는 모든 콘텐츠를 담는 요소(컨테이너)
  • W3C https://www.w3.org/TR/2014/REC-html5-20141028/
